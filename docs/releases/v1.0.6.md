# Wedge v1.0.6

## Highlights

- Add hardcoded custom fields so any custom field can be locked to a fixed value in settings.
- Render hardcoded fields as read-only in the popup while still showing the configured value.
- Always send the configured hardcoded value in payload generation and skip popup validation for those locked fields.

## Notes

- The build artifact for this release is `wedge-v1.0.6.zip`.
